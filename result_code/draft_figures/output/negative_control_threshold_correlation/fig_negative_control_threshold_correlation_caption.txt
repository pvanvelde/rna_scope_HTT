OVERVIEW
----------------------------------------------------------------------------------------------------
This figure tests whether negative control background thresholds correlate with
experimental mRNA expression levels. A LACK of correlation validates that the
experimental signal represents true biological mRNA, not technical artifacts
influenced by background levels.

BIOLOGICAL RATIONALE:
- Negative control thresholds (from DapB probe) represent non-specific background
- Experimental mRNA signal should be independent of background threshold
- If correlated: suggests threshold-dependent bias in quantification
- If uncorrelated: validates that mRNA counts reflect true expression

STATISTICAL ANALYSIS:

  Pearson correlation coefficient (r):
    - Measures strength and direction of linear relationship between two variables
    - Range: -1 (perfect negative) to +1 (perfect positive), 0 = no correlation
    - Computed as: r = cov(X,Y) / (std(X) * std(Y))
    - Interpretation thresholds:
        |r| < 0.3: weak correlation (acceptable)
        |r| 0.3-0.5: moderate correlation (potential concern)
        |r| > 0.5: strong correlation (problematic)

  P-value:
    - Tests null hypothesis that true correlation = 0
    - p < 0.05: correlation is statistically significant
    - p > 0.05: cannot reject null hypothesis (no significant correlation)
    - For this analysis, p > 0.05 is DESIRED (confirms no correlation)

  Regression lines:
    - Ordinary least squares (OLS) linear fit
    - Shows trend direction and magnitude visually
    - Only shown for Q111 (Wildtype n=3 too small for reliable fit)

AGGREGATION:
  - Data aggregated to slide-region level (mean across FOVs)
  - One point per slide per region per channel

NUCLEAR VOLUME NORMALIZATION:

  - mRNA counts normalized per nucleus (not per cell)
  - Accounts for 3D sectioning of nuclei in tissue sections
