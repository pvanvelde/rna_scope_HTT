METHODOLOGY:
--------------------------------------------------------------------------------
Cluster size calculation:
  - Cluster intensity = sum of all spot intensities in a cluster
  - Single spot peak intensity = mode of KDE-fitted intensity distribution
  - Cluster size (mRNA equivalents) = Cluster intensity / Single spot peak
  - This gives the number of mRNA molecules per cluster
  - Normalization is slide-specific to account for batch effects

Cluster detection:
  - DBSCAN clustering algorithm (eps=0.75 µm, min_samples=3)
  - Identifies spatially proximate spots as clusters
  - Only clusters with total intensity > threshold are analyzed
  - Threshold = 2.5 × peak_intensity (slide-specific)
  - Ensures clusters represent genuine aggregation, not noise

Peak intensity determination:
  - Kernel Density Estimation (KDE) on single spot intensities
  - Computed per slide/region/channel combination
  - Accounts for slide-to-slide variation in signal intensity
  - Uses Scott's bandwidth method for optimal smoothing

Quality control:
  - Minimum 5 clusters per FOV required for inclusion
  - Minimum 50 single spots required for peak intensity calculation
  - Slides m1a2 and m1b5 excluded (technical failures)
  - Final filter applied to all spot detections (PFA < 0.05)

Statistics reported (per panel):
  - n: total number of clusters analyzed
  - Mean: arithmetic mean of cluster sizes (mRNA equivalents)
  - Med: median cluster size (50th percentile)
  - IQR: interquartile range (75th - 25th percentile)
  - P95: 95th percentile of cluster sizes
  - cl/cell: average number of clusters per cell (FOV-level mean)
