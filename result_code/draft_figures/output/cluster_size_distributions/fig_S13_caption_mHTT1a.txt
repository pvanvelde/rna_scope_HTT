METHODOLOGY:
--------------------------------------------------------------------------------
Cluster size calculation:
  - Cluster intensity = sum of all voxel intensities in a cluster
  - Single spot peak intensity = mode of KDE-fitted intensity distribution
  - Cluster size (mRNA equivalents) = Cluster intensity / Single spot peak
  - Normalization is slide-specific to account for batch effects

Cluster filtering:
  - CV threshold: CV >= 0.5 (removes low-variance noise)
  - Intensity threshold: > 95th percentile of negative control
  - Minimum 5 valid clusters per FOV for inclusion
