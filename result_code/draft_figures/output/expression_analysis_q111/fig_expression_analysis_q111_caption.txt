METHODOLOGY:
--------------------------------------------------------------------------------
Total mRNA per cell:
  Total = N_spots + (I_cluster_total / I_single_peak) / N_nuclei
  Where I_single_peak is slide-specific peak intensity from KDE

Single mRNA per cell:
  Single = N_spots / N_nuclei
  Represents diffraction-limited individual transcripts

Clustered mRNA per cell:
  Clustered = Total - Single
  Represents aggregated/high-density transcript populations

Peak intensity normalization:
  - Kernel Density Estimation (KDE) with Scott's bandwidth
  - Slide-specific normalization accounts for technical variation
  - Minimum 50 spots required for reliable peak estimation
