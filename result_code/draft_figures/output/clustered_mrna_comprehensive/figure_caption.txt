METHODOLOGY:
--------------------------------------------------------------------------------
mRNA quantification:
  - Single mRNA: Individual diffraction-limited spots (N_spots)
  - Clustered mRNA: Aggregated signal normalized by single-spot peak intensity
  - Total mRNA/nucleus = N_spots + (I_cluster_total / I_single_peak) / N_nuclei
  - Clustered mRNA/nucleus = (I_cluster_total / I_single_peak) / N_nuclei

Normalization:
  - Slide-specific peak intensity determined via Kernel Density Estimation (KDE)
  - Accounts for slide-to-slide variation in signal intensity

Field-of-view (FOV) filtering:
  - Minimum 40 nuclei per FOV (ensures adequate DAPI segmentation)
  - Excludes FOVs with poor tissue quality or imaging artifacts
